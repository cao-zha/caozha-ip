# caozha-ip（IP地址查询）

#### 介绍
caozha-ip，是基于原生PHP写的一套完整的IP转地址模块，支持自动获取IP，也支持查询指定IP，同时支持输出json、jsonp、txt、xml、js等多种IP和地址格式，还可以细分为国家、省、市、地区，方便在各种系统里整合与调用。数据库采用MYSQL，IP地址库来自纯真IP库等。
